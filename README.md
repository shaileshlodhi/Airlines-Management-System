# ✈️ Airlines Management System

This project is a **Java Swing-based desktop application** that helps manage airline operations such as customer handling, flight bookings, ticket generation, and journey management. It is built using **Java**, **Swing GUI**, and **JDBC** for database connectivity.

---

## 📌 Project Objective

To simulate a real-world airline reservation and management system with features like:

- Customer record management
- Flight booking and cancellation
- Ticket generation
- Flight search
- Journey history tracking

---

## 💡 Features

- 🧑‍💼 Add new customer with full details
- 📋 View existing customer list
- 🛫 Book a flight by entering source, destination, and customer ID
- ❌ Cancel an existing flight
- 📄 Generate and print flight tickets
- 🔍 Search flights by destination
- 🧾 View complete flight journey records
